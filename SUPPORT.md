# Support

## How to file issues and get help

This project uses [GitHub issues][gh-issue] to [track bugs][gh-bug], [feature requests][gh-feature], [documentation issues][gh-docs], and [questions or feedback][gh-question]. Please search the existing issues before filing new issues to avoid duplicates. For new topics, see [how to report issues](https://aka.ms/azurelinux-reportissues).

For questions about using this project, please look at the [documentation][docs] and the [How To for Image Customizer][tutorial], or see our [Contributor's Guide][contributor] if you want to work on Azure Linux Image Tools.
